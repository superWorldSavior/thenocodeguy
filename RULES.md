
## Règle : Zéro lien GitHub sur le site

**Règle absolue** : ne jamais ajouter de liens vers GitHub sur thenocodeguy.com.

Interdit :
- Boutons "Voir sur GitHub", "View on GitHub"
- Badges "Open-source on GitHub"
- Liens "Fork on GitHub", "Star on GitHub"
- Toute URL github.com dans les composants JSX/TSX du site
- Mentions "code source disponible" qui pointent vers GitHub

Pourquoi : les repos ne sont pas publics. Ça ne sert à rien et ça crée de la confusion.

S'applique à : tous les composants, pages, articles Lab, guides Workflows, footer, nav, CTA.
