## Description
Personnalité de l'agent admin (rigoureux, proactif sur les deadlines), connecteurs (Gmail/Outlook, stockage fichiers, outil comptabilité), périmètre actions. Livrable: SOUL.md agent + liste connecteurs.

## Acceptance Criteria
Un fichier SOUL.md existe pour l'agent Admin, décrivant sa personnalité (rigoureux, proactif sur les deadlines) et son style de communication. 

Le SOUL.md liste les connecteurs configurés : Gmail/Outlook, stockage fichiers et outil comptabilité, avec l'usage défini pour chacun. 

Le périmètre des actions autorisées est documenté avec des exemples d'actions permises (archivage facture) et interdites (virement bancaire). 

Le document précise les limites légales et responsabilités (l'agent ne valide pas les paiements).
