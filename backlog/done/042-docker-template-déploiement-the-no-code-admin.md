## Description
Docker Compose pour déploiement agent admin. Similaire au Commercial mais avec volumes pour stockage documents clients.

## Acceptance Criteria
Un fichier docker-compose.yml existe pour l'agent Admin et démarre sans erreur avec docker-compose up. 

Le compose définit un volume nommé pour le stockage des documents clients (factures, contrats). 

Un fichier .env.example liste toutes les variables requises (tokens API, chemins volumes, config email). 

La documentation indique la procédure de sauvegarde/restauration des volumes de documents.
