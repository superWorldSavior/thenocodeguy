## Description
Le site ressemble à un template Tailwind générique. Refonte de la typographie (taille, weight, line-height), des espacements entre sections, et de la hiérarchie visuelle pour un rendu plus premium. Pas de changement de couleurs — juste du polish.

## Acceptance Criteria
Hero headline : augmenter la taille (au moins 5xl desktop, bold/extrabold), améliorer le line-height 

Sections : espacement vertical cohérent (padding-y uniforme par type de section) 

Cards agents : moins de padding vide, mieux occuper l'espace, border-radius affiné 

Typographie corps : passer de gray-400 à gray-300 minimum pour améliorer la lisibilité 

CTA primaire : taille légèrement plus grande, padding plus généreux 

Section dividers : lignes ou gradients subtils entre sections (pas de vide brutal) 

Footer : plus structuré, colonnes claires 

Mobile : vérifier que les paddings ne sont pas excessifs sur petits écrans 

Screenshots Playwright avant/après sur homepage desktop+mobile
