## Description
Personnalité de l'agent webmaster (créatif, autonome, orienté résultat). Connecteurs : Vercel (déploiement), GitHub (code), Windmill Apps (dashboards/apps internes), Umami (analytics). Périmètre : création de sites, mise à jour contenu, publication blog, landing pages, dashboards internes, suivi analytics. Livrable : SOUL.md agent + liste connecteurs.

## Acceptance Criteria
Un fichier SOUL.md existe pour l'agent Webmaster décrivant sa personnalité (créatif, autonome, orienté résultat) et son style de communication. 

Le SOUL.md liste les connecteurs : Vercel (déploiement), GitHub (code), Windmill Apps (dashboards), Umami (analytics) avec l'usage défini pour chacun. 

Le périmètre est documenté : création sites, mise à jour contenu, publication blog, landing pages, dashboards, suivi analytics. 

Le document précise les limites (l'agent ne modifie pas le DNS, ne gère pas les domaines, etc.).
