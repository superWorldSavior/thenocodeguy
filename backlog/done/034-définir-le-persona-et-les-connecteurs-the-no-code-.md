## Description
Définir la personnalité de l'agent (ton, style, limites), les connecteurs obligatoires (LinkedIn, Gmail) et optionnels (CRM: Notion, HubSpot, Airtable), et le périmètre des actions autorisées. Livrable: SOUL.md agent + liste connecteurs.

## Acceptance Criteria
Un fichier SOUL.md existe pour l'agent Commercial, décrivant son ton (professionnel, persévérant), son style et ses limites comportementales. 

Le SOUL.md liste les connecteurs obligatoires (LinkedIn, Gmail) avec leur usage défini. 

Le SOUL.md liste les connecteurs optionnels (CRM : Notion, HubSpot ou Airtable) avec conditions d'activation. 

Le périmètre des actions autorisées est documenté avec des exemples explicites d'actions permises et interdites.
