## Description
Créer les pages légales obligatoires pour un site commercial français : mentions légales, CGV (conditions générales de vente pour du service de staffing IA), politique de confidentialité (RGPD). Accessibles depuis le footer.

## Acceptance Criteria
- Page /mentions-legales avec : raison sociale, SIRET, adresse, directeur de publication, hébergeur
- Page /cgv avec : objet du service (placement d'agents IA), modalités, tarification, responsabilités, résiliation
- Page /confidentialite avec : données collectées, finalités, durée de conservation, droits RGPD, contact DPO
- Toutes les pages accessibles depuis le footer
- Traduites en 4 locales (fr, en, zh-TW, zh-CN)
- Build passe sans erreur
