## Description
Préparer le go-to-market : (1) template de message LinkedIn pour la prospection (3 variantes selon la cible), (2) contenu démo montrant un agent en action (vidéo ou screenshots), (3) one-pager PDF de présentation TNCG. Positionnement : "Randstad de l'IA" — on recrute des agents IA pour vos équipes.

## Acceptance Criteria
- 3 templates LinkedIn outreach (PME dirigeant, DRH/COO, freelance/indépendant) dans docs/gtm/
- Chaque template < 300 caractères (limite LinkedIn), avec hook + value prop + CTA
- One-pager PDF (A4) avec : pitch, 3 profils types, process en 4 étapes, contact
- Contenu démo : au minimum 3 screenshots annotés montrant un agent en action (brief → match → résultat)
- Tous les contenus en français, version anglaise en option
