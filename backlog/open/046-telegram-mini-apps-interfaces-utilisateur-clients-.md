## Description
Phase future : déployer des mini-apps Telegram pour donner aux clients des interfaces visuelles (dashboards, formulaires, tableaux de bord) en complément de l'interaction WhatsApp. Priorité basse — à planifier après validation du modèle B2B initial WhatsApp.

## Acceptance Criteria
Un document de spécification des mini-apps Telegram prioritaires existe (au moins 3 cas d'usage identifiés : dashboard analytics, formulaire de demande, tableau de bord activité). 

La roadmap technique précise les prérequis techniques (Telegram Bot API, framework mini-app) et les dépendances avec le modèle B2B WhatsApp. 

Le document définit les critères de déclenchement de cette phase (ex: X clients validés sur le modèle WhatsApp). 

Au moins un prototype ou maquette de mini-app Telegram est fourni comme référence de design.
